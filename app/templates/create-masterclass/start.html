Create a masterclass

A masterclass is a workshop which helps to build digital, data and technology skills in civil servants. Any civil servant can run a masterclass.

To create a new masterclass you will need to provide:

  • a name and description
  • a time and date
  • a location
  • the number of available spaces

Creating a new masterclass takes around 5 minutes.
